# 仓库简介 / Repository Description

阅读卡片知识库：一个把人物、书籍、事件、行业、关系、地图、统计和阅读进度组织成可搜索静态门户的前端系统。适合微信读书笔记、企业家人物志、学习资料库、行业人物库和公开知识产品。

Reading-card knowledge base: a static frontend system for organizing people, books, events, industries, relationships, maps, statistics, and reading progress into a searchable knowledge portal.
